===================================
genai-tag-db-tools ドキュメント
===================================

**genai-tag-db-tools** は、画像生成AI向けのタグ管理ツール。
タグ翻訳やエイリアス管理、使用頻度情報を統合し、一元的なタグデータベース管理を提供。

主な機能
---------
- タグの管理(登録、更新、検索、翻訳管理)
- SQLiteデータベース操作
- PySide6ベースのGUIインタフェース
- 他プロジェクトからのモジュール利用
